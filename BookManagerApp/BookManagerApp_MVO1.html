Book Manager Application

Book Manager Application is a web application to manage purchased books.This app helps to keep track of books that we have purchased. It has search feature to check if you have already purchased a book with the given title. It is a web application built using .NET 8 Razor pages application and a .NET Web API at the backend which interacts with MSSQL db and Azure blob storage.

Technologies used are listed below

  • C# (.NET8)
  • Razor pages application
  • .NET Web API
  • Azure ADB2C
  • Microsoft SQL DB

Features supported are listed below

  • User registration and sign-in
  • Add new book by scanning the ISBN
  • Add new book by providing all the details manually
  • List all the purchased books
  • Search book by title

Application URL: Book Manager App
The application is using the free version of Azure SQL DB, so if the DB has not been accessed for a while, it would not be available. Once we try to login, we access the db and it will take a while to come up, so we will see an exception in this case. I request you to please re-try after 30s and it would be up and you will be able to login.
